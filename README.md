# Vikas Ekka - Portfolio Website

A modern, production-ready developer portfolio website showcasing backend-focused engineering expertise.

## Features

- 🎨 Modern dark theme with clean, professional design
- 📱 Fully responsive layout
- ⚡ Smooth scroll animations
- 📊 Scroll progress indicator
- 🎯 Sticky navigation bar
- 💼 Enterprise-level project showcases
- 📈 Performance optimization case study
- 🎭 Subtle micro-interactions
- ♿ Accessible design with ARIA labels
- 🔍 SEO optimized

## Tech Stack

- **React** - UI library
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **React Icons** - Icon library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

### Update Personal Information

1. **Hero Section**: Edit `src/components/Hero.jsx`
2. **About Section**: Edit `src/components/About.jsx`
3. **Skills**: Edit `src/components/Skills.jsx`
4. **Projects**: Edit `src/components/Projects.jsx`
5. **Experience**: Edit `src/components/Experience.jsx`
6. **Contact**: Edit `src/components/Contact.jsx` and update email/social links

### Update Resume Download

Replace the placeholder resume download functionality in `src/components/Hero.jsx` with your actual resume file.

### Update Social Links

Update all social media links in:
- `src/components/Hero.jsx`
- `src/components/Contact.jsx`
- `src/components/Footer.jsx`
